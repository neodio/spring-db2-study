# 환경
- java 17
- Gradle 8.8
- Spring boot 3.3.2

# Local 개발환경을 위한 Docker
- 실행방법
  ```
  $ cd docker && docker-compose up -d
  ```
- 도커 종료방법
  ```
  $ cd docker && docker-compose down -v
  ```
- h2 console : http://localhost:8081
